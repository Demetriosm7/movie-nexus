MovieNexus is a movie search and watchlist web application built using JavaScript and the OMDb API.

## Features

- Search movies using the OMDb API
- Sort results by newest or oldest release year
- Add movies to a watchlist using localStorage
- Loading spinner during API requests
- Light / Dark theme toggle
- Keyboard support (press Enter to search)

## Technologies Used

- HTML5
- SCSS
- JavaScript (ES6+)
- OMDb API
- LocalStorage

## Future Improvements

- Pagination for search results
- Improved mobile UI
